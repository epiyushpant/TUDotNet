Project Assignment 
Title: Product Management System in Blazor


Objective
Build a Blazor application that demonstrates:
One-way and two-way data binding
Event handling
Navigation and routing
Client-side and server-side validation

Requirements


One-Way Binding : Display a product list (name, price) using one-way binding.
Two-Way Binding : Create a product form with two-way binding for adding/editing products.
Event Handling : Implement buttons for Add, Update, and Delete with @onclick.
Navigation & Routing : Create pages: Home, Products, About.
Use @page directive and NavLink for navigation.


Validation
Apply Data Annotations for client-side validation (e.g., required fields).
Add server-side validation before saving to a database or mock storage.
